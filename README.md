# E-Shop Frontend

Frontend application for the E-Shop Book Management System built with React using TypeScript.

## Technologies

- React
- TypeScript
- Material UI
- Axios

---

## Features

- Browse books, authors, and countries
- Filter books by state 
- JWT authentication support
- Responsive UI
- REST API integration

---

## Run 

Install dependencies:

```bash
npm install
npm run dev
```
Frontend runs on:
http://localhost:3000
